- **Barcode scanner:** Most USB or wireless scanners type numbers into whatever box is active—click in the **barcode** or **search** field on the sale screen first, then scan. The line item should add straight away.
- **Search:** Type part of the product name or code; results filter as you type.
- **Categories:** Tap a category to narrow the product grid; combine with search if the list is long.

**If scanning sometimes fails:** Another app may be “stealing” the keyboard focus, or the cable/connection is loose. On a **tablet with USB adapter**, try reconnecting the scanner. If it still fails, type the code by hand to keep the queue moving, then fix the device when quiet.
